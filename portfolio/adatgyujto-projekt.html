Környezeti Adatgyűjtő Rendszer

Zátrok Márton

1. Feladat Leírása

RP2040-es mikrovezérlő segítségével kiolvastuk a LDR és DHT11 értékeit, amik fényerőt és hőmérsékletet, valamint páratartalom adatokat mértek, és egy fájlba el is mentik az adatokat. Ezt a projekt feladatot egyedül kellett megcsinálnunk, és az elkészítése nagyjából 3 órát vett igénybe.

Kiraktuk az ablakba és elem segítségével működött, ami nem bírta sok ideig, ezért lemerült. Így csak 8 órán át mentette el az adatokat (amit percenként tett meg egyszer), így elég sok adatot kaptam, szépen kivehető a sötétedés mértéke az adatsorban, ahogyan a fényérzékelő mérte a környezeti fényerőt.

Mellékesen mértük és mentettük az elem energiaszintjét is. Egy 9 Voltos elemről indultunk, amit 2,2 Voltra korlátoztunk le a feszültségosztóval. Egészen 1,58 Voltig üzemelt a rendszer; ez volt az a pont, amikor 1,58 alatt már nem volt képes menteni a RP2040 mikrovezérlő, és leállt az adatgyűjtéssel. Szóval 2,23 Voltról egészen 1,58 Voltig működött a rendszer stabilan.

A Wokwi szimulátorban nincs DHT11 csak DHT22 ezért ott az van használva.

2. Komponensek, érzékelők

Lyukacsos panelen lett elkészítve, érkezőket kaptunk ugye és a mikrovezérlőt de a többit nekünk kellet megkeresni ugye kellet kondenzátor 2,2 uF-os 10 K és 20 K ellenállás kellet a feszültség osztónak.

  • Telep: 9V-os elem
  • Ellenállás: 10KΩ, 20KΩ
  • Kondenzátor: 2,2 uf/50V; 2,2 uf/50V
  • DHT11: Hőmérséklet és páratartalom mérő
  • LDR: Fényerő mérő szenzor
  • RP2040: Raspberry Pi RP2040-Zero
  • 7805: Integrált feszültség Stabilizátor

Python Kód: (amit a WokWi-ba töltünk bele)

import machine import utime import dht # Érzékelők inicializálása - DHT11 használata Wokwi kompatibilitás miatt dht_sensor = dht.DHT11(machine.Pin(0)) ldr_adc = machine.ADC(machine.Pin(26)) # RTC inicializálása (Wokwi automatikusan kezeli) rtc = machine.RTC() rtc.datetime((2025, 3, 5, 0, 8, 30, 0, 0)) def read_sensors(): try: dht_sensor.measure() return { 'temp': dht_sensor.temperature(), 'hum': dht_sensor.humidity(), 'light': ldr_adc.read_u16() // 256 # 0-255 skálára konvertálás } except Exception as e: print(f"Érzékelő hiba: {e}") return None while True: sensor_data = read_sensors() if sensor_data: current_time = rtc.datetime() timestamp = f"{current_time[0]}-{current_time[1]:02}-{current_time[2]:02} {current_time[4]:02}:{current_time[5]:02}:{current_time[6]:02}" data_line = f"{timestamp};{sensor_data['temp']:.1f};{sensor_data['hum']:.1f};{sensor_data['light']}\n" # Adatok mentése és megjelenítése with open("eredmenyek.txt", "a") as f: f.write(data_line) print(data_line.strip()) utime.sleep(1) # 1 másodperc szünet (teszteléshez)

Python Kód: (amit az RP2040-be töltünk bele)

import machine from machine import Pin import utime import dht klima_sensor = dht.DHT11(Pin(2)) rtc = machine.RTC() rtc.datetime((2025, 3, 19, 0, 10, 30, 15, 0)) fenyerzekelo = machine.ADC(Pin(25)) feszultseg_meres = machine.ADC(Pin(28)) while True: try: klima_sensor.measure() hofok = klima_sensor.temperature() paratartalom = klima_sensor.humidity() feny = fenyerzekelo.read_u16() // 257 feszultseg = feszultseg_meres.read_u16() // 257 idopont = rtc.datetime() idobelyeg = f"{idopont[0]}-{idopont[1]:02}-{idopont[2]:02} {idopont[4]:02}:{idopont[5]:02}:{idopont[6]:02}" adat_sor = f"{idobelyeg};{hofok:.1f};{paratartalom:.1f};{feny};{feszultseg}\n" with open("Környezeti_adatok.txt", "a") as fajl: fajl.write(adat_sor) print(adat_sor.strip()) utime.sleep(60) except Exception as e: print(f"Rendszerhiba: {e}") utime.sleep(0.6)
