Akasztófa Játék (Tkinter)

Zátrok Márton

1. Feladat kiírása

A projekt feladata egy grafikus felületű Akasztófa (szókirakó) játék készítése Python programozási nyelven, a Tkinter könyvtár segítségével. A programnak egy véletlenszerűen kiválasztott magyar szót kell megjelenítenie csillagokkal elfedve, amelyet a játékosnak betűk tippelésével kell kitalálnia, mielőtt az akasztófa teljesen kirajzolódna.

Felhasználói felület követelményei:

  • „Új játék” gomb: Új szó kiválasztása, hibaszámláló nullázása, rajz törlése, feladvány újragenerálása.
  • Hibák száma kijelzés: Szövegdoboz vagy címke, amely a rossz tippek számát mutatja (Maximum 8 hiba).
  • Feladvány megjelenítése: A kitalálandó szó csillagokkal elfedve jelenik meg (pl. VONAT → VON*T), a helyesen kitalált betűk pedig a megfelelő helyen látszanak.
  • Magyar betűs billentyűzet: Külön gomb minden betűhöz, beleértve az ékezeteseket is. A gombokra kattintva lehessen tippelni.
  • Akasztófa rajz: Vászon (Canvas) elemre rajzolva. Hibánként egy új testrész jelenik meg: 1. fej, 2. törzs, 3. bal kar, 4. jobb kar, 5. bal láb, 6. jobb láb.

Program működése és programozási követelmények:

  • Játék indításakor véletlen szó kiválasztása, csillagok megjelenítése, hibák lenullázása.
  • Tippeléskor a program ellenőrzi a betűt: ha szerepel, megjelenik; ha nem, a hibák száma nő, és a rajz kiegészül.
  • Játék vége: Győzelem (minden betű kitalálva) vagy Vereség (maximális hibaszám elérve). A végén a billentyűzet letiltásra kerül.
  • A program nyelve Python, a felület Tkinter. Objektumorientált (osztály alapú) felépítést, halmazt (vagy listát) kell használni a tippekhez, a kód legyen logikus és kommentelt.

2. Program működésének leírása

A program objektumorientált (OOP) szemlélettel készült, a teljes logikát és a grafikus felületet az Akasztofa osztály fogja össze.

  • Inicializálás és Állapotkezelés: Induláskor a program egy listából véletlenszerűen választ egy feladványt. A felhasználó eddigi tippjeit egy halmaz (self.tippek = set()) tárolja, ami biztosítja, hogy egy betűt csak egyszer lehessen feldolgozni. A hibák számát egy számláló követi nyomon (maximum 8).
  • Felület (felepites metódus): A Tkinter felület Frame-ek (keretek) segítségével van tagolva. Felül található az "Új játék" gomb, a hibaszámláló és a csillagokkal elfedett feladvány (egy Label elemben). Alul bal oldalon helyezkedik el a dinamikusan generált, magyar ékezetes karaktereket is tartalmazó virtuális billentyűzet, jobb oldalon pedig a Canvas (vászon) a rajzoláshoz.
  • Játékmenet (tippeles metódus): Amikor a játékos rákattint egy betűre, a gomb inaktívvá válik (state="disabled"). Ha a betű szerepel a szóban, a gomb zöldre vált, és a szó kijelzője frissül, felfedve a talált karaktereket. Ha a tipp hibás, a gomb pirosra vált, a hibaszámláló nő, és meghívódik a rajzoló metódus.
  • Rajzolás (akasztot_rajzol metódus): Mivel a feladatkiírás 8 hibát engedélyez, de csak 6 testrészt sorol fel, a program az első két hibánál az állványt és a kötelet rajzolja ki piros vonalakkal, majd a 3-8. hibáknál a fejet, törzset és a végtagokat feketével.
  • Játék vége (jatek_vege metódus): Ha a játékos kitalálja a szót, vagy eléri a 8 hibát, egy messagebox (felugró ablak) értesíti az eredményről, megmutatja a helyes szót vereség esetén, és az összes betűgombot letiltja.

3. Program futása (képekkel)

1. Kép: A játék kezdőképernyője

A program induláskor kiválaszt egy szót, elrejti csillagokkal, a hibaszámláló 0-n áll, a gombok aktívak.

2. Kép: Játék közben

A felhasználó tippel. A helyes betűk zöld színnel, a hibásak pirossal jelennek meg a billentyűzeten. A vásznon a hibák számának megfelelően épül az akasztófa.

3. Kép: Játék vége

A játék befejeződött, a program egy felugró ablakban tájékoztat az eredményről, a virtuális billentyűzet gombjai pedig letiltásra kerültek.

➤ Digitális Tartalom: Programkód (Python)

A játék teljes, futtatható forráskódja:

import tkinter as tk from tkinter import messagebox import random class Akasztofa: def __init__(self, root): self.root = root self.root.title("Akasztófa") self.root.geometry("850x500") # Szólista self.szavak = [ "ALMA", "KÖRTE", "SZILVA", "BARACK", "RÁDIÓSTÚDIÓ", "PROGRAMOZÁS", "PYTHON", "BILLENTYŰZET", "MONITOR", "TELEFON", "ISKOLA", "EGYETEM", "TANULÁS", "VIZSGA", "AUTÓ", "REPÜLŐ", "VONAT", "KERÉKPÁR", "UTAZÁS" ] # Játék változók self.kivalasztott_szo = "" self.tippek = set() self.hibak_szama = 0 self.max_hiba = 8 # Grafikus felület felépítése self.felepites() # Első játék indítása self.uj_jatek() def felepites(self): """A grafikus felület elemeinek elhelyezése a kép alapján.""" # --- FELSŐ SÁV (Grid elrendezés) --- felso_keret = tk.Frame(self.root, pady=20, padx=20) felso_keret.pack(fill="x") # Oszlopok konfigurálása a távolságtartáshoz felso_keret.columnconfigure(0, weight=0) # Bal oldal (gomb, felirat) felso_keret.columnconfigure(1, minsize=200) # Üres tér középen (TÁVTARTÓ) felso_keret.columnconfigure(2, weight=0) # Jobb oldal (címkék) felso_keret.columnconfigure(3, weight=0) # Jobb oldal (értékek) # 1. SOR # "Új játék" gomb - Balra igazítva self.uj_jatek_gomb = tk.Button(felso_keret, text="új játék", command=self.uj_jatek, width=10, bg="#f0f0f0") self.uj_jatek_gomb.grid(row=0, column=0, sticky="w") # "hibák száma" felirat - Jobb oldalra igazítva tk.Label(felso_keret, text="hibák száma").grid(row=0, column=2, sticky="e", padx=10) # Hibaszámláló doboz - Jobb szél self.hiba_cimke = tk.Label(felso_keret, text="0", bg="white", width=5, relief="sunken", anchor="center") self.hiba_cimke.grid(row=0, column=3, sticky="w") # 2. SOR # "feladvány" felirat - Balra igazítva (a gomb alatt) tk.Label(felso_keret, text="feladvány").grid(row=1, column=0, sticky="nw", pady=(15, 0)) # A rejtett szó doboza - Jobbra igazítva (a hibaszámláló alatt kezdődik) # columnspan=2, hogy lefedje a hibaszámláló alatti területet is self.szo_kijelzo = tk.Label(felso_keret, text="", font=("Arial", 12, "bold"), bg="white", relief="sunken", width=40, anchor="w") self.szo_kijelzo.grid(row=1, column=2, columnspan=2, sticky="ew", pady=(15, 0), padx=(10, 0)) # --- ALSÓ RÉSZ (Billentyűzet és Rajz) --- also_keret = tk.Frame(self.root) also_keret.pack(fill="both", expand=True, padx=20, pady=20) # Bal oldal: Billentyűzet billentyuzet_keret = tk.Frame(also_keret) billentyuzet_keret.pack(side="left", fill="y", anchor="n") # Billentyűzet kiosztás a kép szerint betuk = [ ["A", "Á", "B", "C", "D", "E", "É", "F", "G", "H", "I", "Í"], ["J", "K", "L", "M", "N", "O", "Ó", "Ö", "Ő", "P", "Q"], ["R", "S", "T", "U", "Ú", "Ü", "Ű", "V", "W", "X", "Y", "Z"] ] self.gombok = {} for sor_index, sor in enumerate(betuk): sor_frame = tk.Frame(billentyuzet_keret) sor_frame.pack() for betu in sor: gomb = tk.Button( sor_frame, text=betu, width=4, height=2, bg="#f0f0f0", command=lambda b=betu: self.tippeles(b) ) gomb.pack(side="left", padx=2, pady=2) self.gombok[betu] = gomb # Jobb oldal: Akasztófa rajz vászon self.canvas = tk.Canvas(also_keret, width=300, height=350, bg="white") self.canvas.pack(side="right", padx=50) def uj_jatek(self): """Játék alaphelyzetbe állítása.""" self.hibak_szama = 0 self.hiba_cimke.config(text=str(self.hibak_szama)) self.tippek.clear() self.kivalasztott_szo = random.choice(self.szavak) self.canvas.delete("all") # Kezdő vonal (föld) kirajzolása, ahogy a képen látszik pirossal # (A feladatkiírás szerint 8 hiba van, de csak 6 testrész, ezért # az első 2 hibánál rajzoljuk az állványt, utána a testet) for gomb in self.gombok.values(): gomb.config(state="normal", bg="#f0f0f0") self.frissit_kijelzo() def frissit_kijelzo(self): """A csillagok frissítése.""" kijelzendo = "" for betu in self.kivalasztott_szo: if betu in self.tippek: kijelzendo += betu else: kijelzendo += "*" self.szo_kijelzo.config(text=kijelzendo) if "*" not in kijelzendo: self.jatek_vege(gyozelem=True) def tippeles(self, betu): """Tipp kezelése.""" if betu in self.tippek: return self.tippek.add(betu) self.gombok[betu].config(state="disabled") if betu in self.kivalasztott_szo: # Helyes tipp self.gombok[betu].config(bg="#90EE90") # Világoszöld self.frissit_kijelzo() else: # Rossz tipp self.hibak_szama += 1 self.hiba_cimke.config(text=str(self.hibak_szama)) self.gombok[betu].config(bg="#FFB6C1") # Világospiros self.akasztot_rajzol() if self.hibak_szama >= self.max_hiba: self.jatek_vege(gyozelem=False) def akasztot_rajzol(self): """Rajzolás a hibák száma alapján (Max 8 hiba).""" h = self.hibak_szama c = self.canvas # Piros vonalak, ahogy a képen látható if h == 1: # Talaj + Oszlop c.create_line(50, 300, 200, 300, width=2, fill="red") # Talaj c.create_line(125, 300, 125, 50, width=2, fill="red") # Oszlop elif h == 2: # Gerenda + Kötél c.create_line(125, 50, 225, 50, width=2, fill="red") # Gerenda c.create_line(225, 50, 225, 80, width=2, fill="red") # Kötél elif h == 3: # Fej (1. testrész a kiírás szerint) c.create_oval(210, 80, 240, 110, width=2, outline="black") elif h == 4: # Törzs (2. testrész) c.create_line(225, 110, 225, 180, width=2, fill="black") elif h == 5: # Bal kar (3. testrész) c.create_line(225, 120, 205, 150, width=2, fill="black") elif h == 6: # Jobb kar (4. testrész) c.create_line(225, 120, 245, 150, width=2, fill="black") elif h == 7: # Bal láb (5. testrész) c.create_line(225, 180, 205, 220, width=2, fill="black") elif h == 8: # Jobb láb (6. testrész) c.create_line(225, 180, 245, 220, width=2, fill="black") def jatek_vege(self, gyozelem): """Játék vége ablak.""" for gomb in self.gombok.values(): gomb.config(state="disabled") if gyozelem: messagebox.showinfo("Játék vége", f"Gratulálok! Kitaláltad a szót: {self.kivalasztott_szo}") else: messagebox.showinfo("Játék vége", f"Sajnálom, vesztettél! A megfejtés: {self.kivalasztott_szo}") self.szo_kijelzo.config(text=self.kivalasztott_szo) if __name__ == "__main__": root = tk.Tk() app = Akasztofa(root) root.mainloop()

Elég sokat kellett játszani a grid és a pack beállításokkal, mire a billentyűzet és a rajzolós rész (Canvas) normálisan illeszkedett egymás mellé.

Volt a feladatban egy kis csavar is: 8 hibát lehetett véteni, de csak 6 testrész volt felsorolva. Ezt végül úgy oldottam meg, hogy az első két hibánál még csak az akasztófa állványát rajzolom ki pirossal, és csak utána jön a fekete pálcikaember. Szerencsére az osztályok (OOP) használata sokat segített, így nem lett egy nagy káosz a kód, és könnyebb volt nyilvántartani a tippeket. Összességében örülök a végeredménynek, mert a program stabilan működik, és sikerült mindent beletennem, amit a kiírás kért."
