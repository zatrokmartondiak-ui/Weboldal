Palackozó Gépsor Hőmérséklet-Szabályozó Rendszer

Zátrok Márton

1. A projekt célja és feladata

A projekt célja egy ipari palackozó gépsor pasztőrözési szakaszához tartozó automatizált hőmérséklet-szabályozó rendszer elektronikai és szoftveres részének megtervezése és megépítése. A rendszer feladata a pasztőrözési folyamat során előírt hőmérsékleti profil pontos és megbízható követése, valamint a kezelő számára a szükséges információk megjelenítése.

A megvalósított modell egy víztartályból és egy fűtő elemből áll, ahol a víz hőmérsékletét egy DS18B20 típusú digitális hőmérséklet-érzékelő figyeli. A vezérlést egy RP2040 mikrovezérlő végzi. A rendszer alfanumerikus kijelzőn mutatja az aktuális értékeket, nemcsak fűtéskor, hanem hűléskor is.

2. Technikai megvalósítás

A folyamat az alábbi lépésekből áll:

  • A hideg munkadarab hőmérsékletének emelése 60 °C-ra, majd tartás 30 másodpercig.
  • Lehűtés, amíg a hőmérséklet 30°C alá nem csökken, ekkor készenlétbe áll.

A működtetéshez folyamatindító gomb és vészleállító gomb is rendelkezésre áll. Fontos szempont volt a szabályozási pontosság és a biztonság.

Alkatrészjegyzék:

  • Mikrovezérlő: RP-2040 Zero
  • Kijelző: LCD 16x2 (alfanumerikus)
  • Szenzor: DS18B20 Hőmérséklet érzékelő
  • Beavatkozó: Fűtőszál kapcsoló (Relay)
  • Gombok: Indítás (Start), Vészleállító (Stop)
  • Passzív elemek: 1kΩ ellenállások (LCD-hez és gombokhoz)

➤ Digitális Tartalom: Python Kód (MicroPython)

A vezérlő programkódja:

from machine import Pin import onewire import ds18x20 import time # === Hőmérséklet szenzor inicializálása === ds_pin = machine.Pin(3) ow = onewire.OneWire(ds_pin) ds_sensor = ds18x20.DS18X20(ow) roms = ds_sensor.scan() if not roms: raise Exception("Nincs DS18B20 szenzor csatlakoztatva!") sensor_rom = roms[0] # === Fűtőelem kimenet === heater = Pin(16, Pin.OUT, value=1) # === LCD (16x2) === class Lcd: def __init__(self, rs, en, d4, d5, d6, d7): self.rs = Pin(rs, Pin.OUT) self.en = Pin(en, Pin.OUT) self.data = [Pin(d, Pin.OUT) for d in [d4, d5, d6, d7]] self.cmd(0x33) time.sleep_ms(5) self.cmd(0x32) time.sleep_ms(5) self.cmd(0x28) self.cmd(0x0C) self.cmd(0x06) self.cmd(0x01) time.sleep_ms(5) def pulse(self): self.en.off() time.sleep_us(1) self.en.on() time.sleep_us(1) self.en.off() time.sleep_ms(2) def send(self, value, mode): self.rs.value(mode) for i in range(4): self.data[i].value((value >> (4 + i)) & 1) self.pulse() for i in range(4): self.data[i].value((value >> i) & 1) self.pulse() def cmd(self, cmd): self.send(cmd, 0) def write(self, text): for char in text: self.send(ord(char), 1) def clear(self): self.cmd(0x01) time.sleep_ms(5) def move_to(self, x, y): addr = 0x80 + x + 0x40 * y self.cmd(addr) lcd = Lcd(12, 11, 10, 9, 8, 7) # === Gombok (START a Pin(1), VÁSZLEÁLLÍTÓ a Pin(2)) === start_btn = Pin(1, Pin.IN, Pin.PULL_UP) stop_btn = Pin(2, Pin.IN, Pin.PULL_UP) # === Állapot === fut = False def meres(): global fut if fut: return fut = True lcd.clear() lcd.write("Folyamat indul") time.sleep(1) def aktualis_homerseklet(): ds_sensor.convert_temp() time.sleep_ms(750) return ds_sensor.read_temp(sensor_rom) def varj_celig(cel, timeout_ms): start = time.ticks_ms() while True: t = aktualis_homerseklet() print("T:", t) lcd.move_to(0, 1) lcd.write("T: {:.1f} C ".format(t)) if t >= cel: return True if not stop_btn.value(): lcd.clear() lcd.write("Veszleallas!") heater.on() return False if time.ticks_diff(time.ticks_ms(), start) > timeout_ms: lcd.clear() lcd.write("Timeout!") heater.on() return False heater.off() # Relé Bekapcsolása def tarts(masodperc): start = time.ticks_ms() while time.ticks_diff(time.ticks_ms(), start) < masodperc * 1000: t = aktualis_homerseklet() lcd.move_to(0, 1) lcd.write("T: {:.1f} C ".format(t)) if not stop_btn.value(): lcd.clear() lcd.write("Veszleallas!") heater.on() return False time.sleep(1) return True # Felfűtés 40°C lcd.clear() lcd.write("Felfutes 40C") heater.off() if not varj_celig(40, 300000): fut = False return # Tartás 50s lcd.clear() lcd.write("Tartas 50s") if not tarts(50): fut = False return # Felfűtés 70°C lcd.clear() lcd.write("Felfutes 70C") heater.off() if not varj_celig(70, 300000): fut = False return # Tartás 20s lcd.clear() lcd.write("Tartas 20s") if not tarts(20): fut = False return # Hűlés heater.on() lcd.clear() lcd.write("Hules...") start = time.ticks_ms() while True: t = aktualis_homerseklet() lcd.move_to(0, 1) lcd.write("T: {:.1f} C ".format(t)) if t < 30: break if not stop_btn.value() or time.ticks_diff(time.ticks_ms(), start) > 600000: lcd.clear() lcd.write("Hules leallt") fut = False return time.sleep(1) lcd.clear() lcd.write("Kesz!") fut = False # === Főciklus === lcd.clear() lcd.write("Keszenlet") while True: if not start_btn.value() and not fut: meres() lcd.clear() lcd.write("Keszenlet") time.sleep_ms(100)

4. Önreflexió

"A projekt megvalósítása nem volt nehéz, mert már csináltunk feladatokat a gombokkal és a kijelzővel is. Ami ismeretlen volt, az a Relay és a DS18B20; mind a kettőt könnyedén bekötöttem, azzal nem volt gond. Viszont a Wokwinak baja volt a DS18B20 modullal, több idő telt el, mire rájöttem, hogy ez csak a micropython-20231005-v1.21.0 verzióban működik, ezért átírtam arra a verziószámot, és működött minden, ahogy kell. Lehet, azért volt ez a verzióprobléma, mert az érzékelő még béta verzióban volt. Na mindegy is, a lényeg, hogy minden könnyedén összeállt, ahogy kellett, és működött minden."
